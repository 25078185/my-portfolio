##  Project Overview
This is a responsive personal portfolio built from scratch as part of the COM 2301 Human‑Computer Interaction module. The site showcases my academic journey, technical skills, and GitHub projects while adhering to core HCI design principles.

 Features
- **Responsive Design** — Optimised for mobile, tablet, and desktop viewing using CSS Grid, Flexbox, and media queries.
- **Dynamic Projects Section** — Fetches live repository data from the GitHub API and highlights the most relevant projects using a custom scoring algorithm.
- **Interactive Elements** — Multi‑line typing animation, spinning gradient border on profile photo, and smooth hover effects.
- **HCI Best Practices** — Consistent navigation feedback, clear affordances, readable typography, and accessible touch targets.
- **Contact Integration** — Direct email links and a `mailto` contact form.
